# 银河火线隐私政策

**更新日期：2026年6月3日**

**生效日期：2026年6月3日**

欢迎您使用《银河火线》。盘呈工作室（以下简称“我们”）非常重视用户个人信息和隐私保护。本隐私政策用于说明我们在您使用《银河火线》（以下简称“本游戏”）时，如何收集、使用、保存、共享和保护您的个人信息，以及您如何管理您的个人信息。

在使用本游戏前，请您仔细阅读并充分理解本隐私政策。您开始使用本游戏，即表示您已了解并同意本隐私政策。若您不同意本隐私政策的内容，请停止使用本游戏。

如您是未成年人，请您和您的监护人共同阅读本隐私政策，并在取得监护人同意后使用本游戏。

## 一、我们如何收集和使用您的个人信息

### 1. 游戏基础服务

当前版本中，本游戏不自建账号系统，不要求玩家注册游戏内账号，不主动收集玩家昵称、头像、手机号、身份证件号码、IP 地址、设备型号、系统版本、Android ID、OAID、IMEI、IMSI、MAC 地址等个人信息或设备信息。

本游戏会在玩家设备本地保存必要的游戏数据，用于提供游戏进度记录和基础游戏体验。当前本地保存内容包括升级进度、钻石等游戏进度数据。上述数据存储在玩家设备本地，不上传至我们的服务器。

### 2. TapTap 登录

当前版本支持 TapTap 登录。为实现登录、账号识别和 TapTap 相关服务，TapTap 可能会处理您的 TapTap 用户标识、登录状态、账号必要信息，以及系统版本、设备型号、网络类型、AndroidID 等必要设备信息。具体信息处理规则请以 TapTap 或 TapSDK 的隐私政策、平台规则及实际授权页面为准。

本游戏不自建保存您的 TapTap 账号密码、手机号等账号敏感信息。

### 3. 实名认证和防沉迷

根据网络游戏相关法律法规及监管要求，当前版本接入 TapTap 实名认证/防沉迷服务。为完成实名认证、防沉迷判断和合规管理，TapTap 或其合作认证服务方可能会处理您的姓名、身份证件信息、账号标识，以及系统版本、设备型号、网络类型、AndroidID 等必要设备信息。

上述实名认证和防沉迷信息由 TapTap 或相关服务方按其规则处理，本游戏不自建服务器保存您的姓名、身份证件号码等实名信息。如您拒绝提供或无法完成实名认证，可能无法正常登录或使用本游戏，或在游戏时长、使用范围等方面受到相应限制。

### 4. 充值、广告、统计和崩溃分析

当前版本不提供充值、内购、付费 DLC 或广告变现功能，不接入支付 SDK、广告 SDK、统计分析 SDK、崩溃分析 SDK。

项目中虽然存在广告 SDK 接入文档和相关备份文件，但当前版本代码中的广告功能为禁用状态，不启用广告 SDK。若后续版本新增广告、支付、统计、崩溃分析等功能，我们会在隐私政策中更新相应说明，并在符合法律法规要求的前提下使用。

## 二、设备权限调用情况

当前版本不主动申请相机、麦克风、定位、通讯录、短信、通话记录、外部存储、读取手机状态、蓝牙、通知、查询安装应用、安装应用等 Android 敏感权限。为实现 TapTap 登录、实名认证/防沉迷服务，TapTap 相关服务可能使用网络权限、获取网络状态等必要权限或信息，具体以 TapTap 或 TapSDK 的隐私政策及实际上架包权限为准。

若后续版本因新增功能需要申请设备权限，我们会在申请权限前说明权限用途，并在隐私政策中更新相关内容。您可以通过设备系统设置关闭或管理相应权限。关闭权限可能会导致对应功能无法使用，但不影响其他无需该权限的功能。

## 三、我们如何保存您的信息

当前版本不自建服务器存储玩家个人信息。游戏进度、设置、钻石等本地游戏数据保存在玩家设备本地，保存至您主动删除本地存档、清除应用数据或卸载游戏为止。

涉及 TapTap 登录、实名认证/防沉迷的信息，由 TapTap 或相关服务方按照其隐私政策、平台规则及法律法规要求保存和处理。

如未来我们开始自建服务器或上传玩家数据，我们会在隐私政策中说明收集范围、使用目的、保存期限和用户权利行使方式。

## 四、我们如何共享、转让、公开披露您的个人信息

当前版本除 TapTap 登录、实名认证/防沉迷等您主动使用或合规所必需的第三方服务外，我们不会向其他第三方共享玩家个人信息。

我们不会将您的个人信息转让给任何公司、组织或个人，但在发生合并、收购、资产转让、破产清算等情形且涉及个人信息转让时，我们会依法向您告知相关情况，并要求新的个人信息处理方继续受本隐私政策约束。

我们不会主动公开披露您的个人信息，但法律法规、监管要求、司法机关或行政机关依法要求的情形除外。

根据相关法律法规，在以下情形中，我们可能会在无需另行征得您授权同意的情况下处理、共享、转让或公开披露相关信息：

1. 与我们履行法律法规规定的义务相关的；
2. 与国家安全、国防安全直接相关的；
3. 与公共安全、公共卫生、重大公共利益直接相关的；
4. 与刑事侦查、起诉、审判和判决执行等直接相关的；
5. 出于维护您或其他个人的生命、财产等重大合法权益但又很难取得本人同意的；
6. 您自行向社会公众公开的个人信息；
7. 从合法公开披露的信息中收集个人信息的；
8. 法律法规规定的其他情形。

## 六、我们如何保护您的信息

我们非常重视信息安全，并采取合理可行的安全保护措施，防止信息遭到未经授权访问、披露、篡改、丢失或毁坏。

请您理解，互联网环境并非绝对安全。我们会尽力保护您的信息安全，但无法保证任何信息传输或存储过程百分之百安全。

如发生个人信息安全事件，我们将按照法律法规要求及时采取补救措施，并通过适当方式告知您事件基本情况、可能影响、我们已采取或将采取的处置措施、您可采取的防范措施等。

## 九、本隐私政策如何更新

我们可能会根据法律法规变化、产品功能变化、第三方服务变化等情况更新本隐私政策。更新后的隐私政策会在相关页面发布，并以适当方式提醒您。

以下情况发生时，我们会同步更新隐私政策：

1. 新增或删除 Android 权限；
2. 新增 TapTap 登录、实名、防沉迷、支付、广告、统计、崩溃分析、云存档、排行榜等功能；
3. 新增、删除或升级第三方 SDK，且 SDK 收集信息或用途发生变化；
4. 开始上传本地存档、设备信息、日志或用户行为数据；
5. 运营主体、联系方式、隐私政策生效日期发生变化。

未经您明确同意，我们不会削减您按照本隐私政策所享有的权利。
